La fonction busy_cpu() permet de simuler du temps de traitement à l'aide d'une boucle. Cette version exploite la librairie rtlog pour permettre de visualiser l'évolution du système.

void busy_cpu ( int  ms)

Fonction permettant de simuler de l'utilisation de temps CPU.

Cette fonction peut être appelée par plusieurs tâches de manière concurrente. Chaque tâche consommera ensuite le temps CPU demandé.

Paramètres
msNombre de millisecondes à consommer
